Dans ce projet, j'ai utilisé React.Js pour construire une application web moderne et facile à utiliser, qui présente mes compétences en tant que développeur web. Ce projet présente une fonctionnalité de traduction qui permet aux utilisateurs de basculer entre la version française et anglaise ce qui lui permet de toucher un public international.

Pour implémenter cette fonctionnalité, j'ai utilisé les Hooks de React, notamment useState et useEffect.

Les Hooks sont une fonctionnalité de React qui permet aux développeurs d'utiliser les concepts de l'état et des effets dans les composants fonctionnels, plutôt que de devoir utiliser des classes. useState permet de gérer l'état local d'un composant, tandis que useEffect permet de gérer les effets de côté, comme les appels API ou les modifications du DOM.

Pour gérer la traduction, j'ai utilisé useState pour stocker le paramètre de traduction dans l'état local du composant. Lorsque l'utilisateur clique sur le bouton de traduction, la fonction setTranslate met à jour l'état, et j'ai utilisé localStorage pour stocker cette valeur afin que l'utilisateur puisse conserver sa préférence de traduction entre les différentes pages et après avoir fermé le navigateur.

J'ai également utilisé une API externe, TheMovieDB, pour afficher les détails des films dans un composant sidebar affiché dans les articles de l'application.
Pour cela, un service (MoviesService) a été créé pour gérer les appels API, et un composant a été défini pour afficher les données reçues (SidebarMovie).

J'ai utilisé useEffect pour gérer les appels API de manière efficace, de sorte que les données ne soient requêtées qu'une seule fois lorsque le composant est monté.

Enfin, il y a une liste de diffusion gérée par Mailchimp pour permettre aux utilisateurs de s'inscrire pour recevoir des mises à jour sur mon travail et mes projets futurs.

Pour cela, j'ai utilisé une bibliothèque tierce, react-mailchimp-subscribe, qui permet de gérer les formulaires d'inscription à une liste de diffusion Mailchimp en utilisant React. Cette bibliothèque offre une interface simple pour les formulaires d'inscription, avec des fonctionnalités telles que la gestion des états et la possibilité de personnaliser les messages de réponse. J'ai également utilisé les variables d'environnement pour stocker les informations de connexion à ma liste de diffusion Mailchimp, garantissant ainsi la sécurité des données de mes utilisateurs.
