WorstCompany

WorstCompany est un script d'extraction de données qui permet de récupérer une liste d'entreprises triées par note la plus basse depuis Google Maps, en utilisant la bibliothèque de navigation automatisée Puppeteer.

Ce script permet de naviguer automatiquement sur Google Maps, de rechercher des entreprises selon les arguments passés, de récupérer les informations de ces entreprises, telles que le nom, l'adresse, la note, le nombre de commentaire, le lien vers la page de l'entreprise et de les trier par ordre de note décroissante.

Enfin, il génère un fichier CSV contenant toutes les informations collectées.
